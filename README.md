A modern full-stack registration form website created for the She Can Foundation Internship Selection Task.

##  Project Overview

This project is a responsive full-stack web application built using:

- React JS
- Node JS
- Express JS
- MySQL

The website includes:

- NGO themed landing page
- Modern UI inspired by the official She Can Foundation website
- Registration form
- Backend API integration
- MySQL database connectivity

Submitted form data is stored directly in the MySQL database.

---

##  Features

- Responsive full-screen design
- Professional NGO-style landing page
- React frontend
- Express backend
- MySQL database integration
- Form submission handling
- Modern UI/UX
- Fully functional registration system
